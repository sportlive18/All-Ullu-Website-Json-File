# All Ullu Website JSON Data

This repository contains automated scraped JSON data files containing metadata for various video contents. 

## ⚠️ Disclaimer
**I do not host any of the links, media, or video files provided in these JSON datasets.** 
This project solely acts as an open-source indexer that scrapes publicly available metadata from third-party websites. This repository contains data relating to **18+ content**. Please proceed with discretion.
